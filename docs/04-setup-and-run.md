## Thiết lập & chạy local (React + Vite + TypeScript)

### Yêu cầu
- Node.js LTS (khuyến nghị 18+ hoặc 20+)
- npm/pnpm/yarn (tùy chọn)

### Vị trí source code
Frontend được đặt trong thư mục `web/`.

### Cài dependencies

```bash
cd web
npm install
```

### Chạy local

```bash
cd web
npm run dev
```

### Build

```bash
cd web
npm run build
npm run preview
```

### Cấu hình môi trường
Tạo `.env` theo hướng dẫn trong `docs/05-configuration.md`.
